FibOwen¶

Usage

use FibOwen;
config var args: int = 10¶

Owen Plambeck 5-29-2019 Fibonacci Sequence Program using slow algorithm

proc fib(args: int): int¶

fib proc takes in and int as an argument and if the arg is greater than one recursivly calls back to compute fib sequence. Else it returns the args, which is written out.
